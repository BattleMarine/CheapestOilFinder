# CheapestOilFinder

CheapestOilFinder는 현재 위치나 목적지를 기준으로 주변 주유소를 찾아서, 유가와 이동 거리, 할인 정보를 함께 고려해 더 경제적인 주유소를 고르는 Android 앱입니다.  
이 저장소는 Android 프론트엔드이며, 주유소 데이터 수집과 정규화, DB 저장, 검색 처리는 별도의 Spring 백엔드가 담당합니다.

## 무엇을 해결하나요

- 현재 위치 주변의 주유소를 빠르게 탐색합니다.
- 목적지 또는 경로 주변의 주유소도 함께 비교합니다.
- 주유소 가격만 보지 않고 이동 거리와 할인까지 반영해 비교할 수 있게 합니다.
- 사용자가 지도에서 주유소 위치를 보고, 외부 지도 앱으로 길찾기를 이어갈 수 있게 합니다.

## 핵심 사용 흐름

1. 사용자가 현재 위치 또는 목적지를 입력합니다.
2. Android 앱이 백엔드 API를 호출합니다.
3. 백엔드는 DB에 저장된 최신 주유소와 유가 데이터를 조회합니다.
4. 백엔드는 거리, 유종, 할인, 추천 기준을 반영한 응답을 돌려줍니다.
5. Android 앱은 받은 데이터를 지도 마커와 목록으로 보여줍니다.
6. 사용자는 원하는 주유소를 선택해 카카오맵 등 외부 지도 앱으로 길찾기를 이어갑니다.

## 프론트엔드와 백엔드의 역할

### Android 프론트엔드

- 현재 위치 또는 목적지 입력 UI를 제공합니다.
- Kakao Map SDK로 지도와 마커를 표시합니다.
- 백엔드 API를 호출해 주유소와 추천 결과를 받아옵니다.
- 받은 응답을 목록, 지도, 상세 화면으로 렌더링합니다.
- 외부 지도 앱으로 길찾기 진입점을 제공합니다.
- 현재 위치 화면과 목적지 화면의 GPS 버튼은 기기 위치를 읽어 지도 중심과 현재 위치 마커를 갱신합니다.

### Spring 백엔드

- 오피넷 API를 호출해 주유소와 유가 데이터를 수집합니다.
- PostgreSQL/PostGIS에 주유소와 섹터 데이터를 저장합니다.
- 주변 주유소 검색, 경로 주변 검색, 추천 계산을 제공합니다.
- Android 앱이 직접 오피넷 API를 호출하지 않도록 중간 계층 역할을 합니다.

## 지도와 마커

이 앱은 Kakao Map SDK를 사용합니다.  
지도 화면은 현재 위치 화면과 목적지 화면에 있으며, 주유소 마커와 현재 위치 마커는 분리해서 관리합니다.

- 지도 제어는 `KakaoMapController`가 담당합니다.
- 현재 위치 마커와 주유소 마커는 별도로 유지합니다.
- 지도 렌더링과 마커 갱신은 지도 SDK 전용 타입에 너무 깊게 묶지 않도록 분리해 둡니다.
- 현재 위치 마커는 전용 라벨 레이어에서 파란 점으로 표시하고, 주유소 마커는 그 아래 라벨 레이어에서 표시합니다.
- 현재 위치 라벨 레이어는 경쟁하지 않도록 설정해서, 지도의 다른 라벨과 겹쳐도 우선 보이게 둡니다.
- 현재 위치 화면에서는 GPS 위치를 얻는 즉시 현위치 마커를 먼저 찍고, 그 다음 주유소를 다시 조회합니다.
- 현재 위치와 주유소 마커는 런타임에서 만든 비트맵 스타일을 사용해 그립니다.
- 현재 위치 마커와 주유소 마커는 각각 별도 라벨 레이어에 올리고, `setVisible(true)`와 `showAllLabel()`을 함께 써서 숨김 상태를 막습니다.

## 백엔드 연동

프론트엔드의 주유소 조회는 Retrofit 기반 `BackendStationRepository`를 통해 수행합니다.  
API 계약, 요청/응답 필드, 호출 예시는 아래 문서에 정리해 두었습니다.

- [프론트엔드-백엔드 API 가이드](docs/FRONTEND_BACKEND_API_GUIDE.md)

기본 설정은 Android Emulator에서 `http://10.0.2.2:8080/`를 바라보도록 되어 있습니다.

## 비밀정보 관리

실제 키와 토큰은 Git에 올리지 않습니다.  
로컬 전용 민감 정보는 `secrets/` 폴더에 두고, 앱은 파일을 읽어 값을 주입받습니다.

- 카카오 네이티브 앱 키 파일: `secrets/kakao_native_app_key.txt`
- 예시 파일: `secrets/kakao_native_app_key.txt.example`, `secrets/kakao.properties.example`
- 실제 비밀 파일은 Git에 포함하지 않습니다.

## 현재 상태 메모

- 앱은 Kotlin 기반으로 구현되어 있습니다.
- 현재 위치 화면과 목적지 화면이 준비되어 있습니다.
- Kakao Map SDK 연동이 들어가 있습니다.
- 백엔드 API 호출은 Retrofit 기반 저장소 계층으로 분리되어 있습니다.
- 앱 시작 시 위치 권한을 확인하고, 없으면 요청합니다.
- 현재 위치 화면과 목적지 화면의 GPS 버튼은 기기 위치를 읽어 지도 중심과 현재 위치 마커를 갱신합니다.
- 현재 위치 마커는 파란 점 아이콘을 사용하고, 주유소 마커보다 위 레이어에 표시합니다.
- 현재 위치 레이어는 `CompetitionType.None`으로 두고, 주유소 레이어보다 높은 zOrder를 사용합니다.
- 현재 위치 화면의 GPS 갱신은 위치 수신 즉시 현위치 마커를 먼저 표시하고, 뒤이어 백엔드 주유소 조회를 수행합니다.
- 현재 로컬 백엔드는 문서 기준으로 `GET /api/stations/nearby`를 사용하며, 경로 조회와 상세 조회는 백엔드 배포 상태에 따라 다를 수 있습니다.
