# CheapestOilFinder

CheapestOilFinder는 현재 위치나 목적지를 기준으로 주변 주유소를 찾고, 최저가 주유소를 고르는 Android 앱입니다.  
앱은 Kakao Map SDK를 사용해 주유소 위치를 지도에 표시하고, 백엔드 서버가 제공하는 주유소/유가 데이터를 받아 화면에 보여줍니다.

## 프로젝트가 해결하는 문제

- 현재 위치나 목적지 주변의 주유소를 빠르게 찾고 싶습니다.
- 유가와 거리, 할인 정보를 함께 비교하고 싶습니다.
- 오피넷 API를 앱이 직접 호출하지 않고, 백엔드가 대신 관리하길 원합니다.
- 개발 중에는 네트워크가 자주 바뀌어도 앱이 백엔드에 쉽게 다시 붙어야 합니다.

## 핵심 동작 흐름

1. 사용자가 현재 위치 또는 목적지를 입력합니다.
2. Android 앱이 백엔드 API를 호출합니다.
3. 백엔드는 DB에 저장된 주유소와 유가 정보를 반환합니다.
4. Android 앱은 받은 데이터를 지도 마커와 목록으로 표시합니다.
5. 사용자는 주유소를 선택하고 외부 지도 앱으로 길찾기를 이어갈 수 있습니다.

## 프론트엔드와 백엔드 역할

### Android 앱

- 현재 위치와 목적지를 입력받습니다.
- Kakao Map SDK로 주유소를 지도에 보여줍니다.
- 백엔드 API를 호출해 주변 주유소와 유가 정보를 받습니다.
- 받은 데이터를 지도 마커, 유가 박스, 목록으로 렌더링합니다.
- 외부 내비게이션 앱으로 이동하는 진입점을 제공합니다.

### 백엔드 서버

- 오피넷 API를 대신 호출합니다.
- DB에 주유소와 유가 데이터를 저장합니다.
- 주변 주유소 검색과 경로 주변 주유소 검색을 제공합니다.
- 최신 유가와 섹터 갱신 상태를 관리합니다.

## 백엔드 연동

프론트엔드는 백엔드 REST API를 호출합니다.  
세부 요청/응답 형식은 아래 문서를 참고합니다.

- [프론트엔드-백엔드 API 가이드](docs/FRONTEND_BACKEND_API_GUIDE.md)

기본 개발 주소는 Android Emulator 기준 `http://10.0.2.2:8080/`입니다.  
실기기 테스트나 네트워크가 자주 바뀌는 환경에서는 백엔드 주소를 쉽게 바꿀 수 있어야 합니다.

## 백엔드 주소를 쉽게 바꾸는 방법

이 프로젝트는 백엔드 주소를 코드에 박아두기보다 **설정값으로 바꾸는 방식**을 권장합니다.

### 권장 우선순위

1. 앱 설정 화면에서 주소를 직접 입력
2. 로컬 전용 설정 파일로 기본 주소 주입
3. 설정이 없으면 개발 기본값 사용

### 로컬 개발 예시

- 에뮬레이터: `http://10.0.2.2:8080/`
- 같은 Wi-Fi 실기기: `https://...` 형태의 Cloudflare Tunnel 주소 또는 같은 네트워크의 고정 주소
- USB 연결 실기기: `adb reverse tcp:8080 tcp:8080`

### Cloudflare Tunnel을 쓰는 경우

도메인을 아직 구매하지 않아도 Cloudflare Tunnel의 Quick Tunnel을 쓰면 임시 HTTPS 주소를 받을 수 있습니다.

- 주소는 실행할 때마다 바뀝니다.
- 터널이 살아 있는 동안만 사용할 수 있습니다.
- 테스트용으로는 편하지만, 반복 사용 시에는 앱의 백엔드 주소를 쉽게 바꿀 수 있어야 합니다.

즉, Quick Tunnel을 쓰는 개발 흐름에서는:

1. 백엔드 서버 실행
2. Cloudflare Tunnel 실행
3. 터미널에 나온 HTTPS 주소를 앱의 백엔드 주소 설정에 붙여넣기
4. 저장 후 바로 호출

이 흐름이 가장 편합니다.

## 현재 상태 메모

- 앱은 Kotlin 기반으로 구성되어 있습니다.
- Kakao Map SDK 연동이 들어가 있습니다.
- 백엔드 API 호출은 Retrofit 기반 저장소 계층으로 분리되어 있습니다.
- 현재 위치 화면과 목적지 화면이 존재합니다.
- 현재 위치 화면은 지도가 화면 전체를 채우고, 뒤로가기와 GPS, `현 위치로 설정` 버튼이 지도 위에 겹쳐 보이도록 구성합니다.
- 목적지 화면도 현재 위치 화면과 같은 풀스크린 지도 오버레이 구조를 사용합니다.
- 현재 위치 화면은 GPS를 읽어 현재 위치 마커를 표시하고, 반경 주유소를 백엔드에서 받아옵니다.
- 현재 위치 화면에 들어가면 즉시 주변 주유소를 자동으로 불러와 리스트와 마커를 보여줍니다.
- GPS 버튼은 리스트와 분리된 고정 위치에 항상 보이며, 누르는 즉시 캐시된 현재 위치로 먼저 이동하고, 재수신 성공 시 현재 위치 마커로 다시 한 번 중심을 맞춥니다. 위치가 10m 이상 바뀐 경우에만 다시 주유소를 요청합니다.
- GPS 버튼은 지도 위에 보이되, 주유소 리스트나 정보 패널이 올라오면 그 아래로 가려지도록 레이어를 조정했습니다.
- 현재 위치 화면 오른쪽 중앙에는 `+`/`-` 줌 버튼이 있고, 이 버튼은 주유소가 5개, 20개, 50개, 전체로 보이는 4단계 프리셋만 이동합니다.
- 줌 버튼은 GPS 버튼과 같은 크기로 맞춰 통일했고, 바로 위에 붙어 있으며 두 버튼 사이에는 약간의 간격을 둡니다.
- 손가락 핀치는 기존처럼 자유롭게 확대/축소할 수 있습니다.
- 주유소 마커와 유가 말풍선은 화면 아래쪽에 있는 항목이 위로 더 잘 쌓이도록 화면 기준으로 정렬합니다.
- 주유소 가격 박스는 0원인 유종을 숨기고, 보통휘발유와 고급휘발유가 모두 있으면 `휘발유(고급) 보통가격(고급가격)` 형식으로 함께 표시합니다.
- 보통휘발유만 있으면 `휘발유 보통가격`, 고급휘발유만 있으면 `고급휘발유 고급가격` 형식으로 표시합니다.
- 같은 Wi-Fi 환경의 실기기 테스트를 위해 백엔드 주소를 쉽게 바꿀 수 있는 구조를 권장합니다.

## 현재 위치 화면 동작 메모

- 현재 위치 화면의 주유소 마커는 줌 레벨에 따라 유가 박스 표시 수준이 달라집니다.
- 확대가 충분하면 보통휘발유, 고급휘발유, 디젤을 모두 표시합니다.
- 어느 정도 줌아웃하면 디젤을 숨기고 휘발유 계열만 표시합니다.
- 더 줌아웃하면 유가 텍스트 박스를 모두 숨기고 주유소 아이콘만 표시합니다.
- `현재 위치로 설정` 버튼을 누르면 반경 5km 주유소가 하단 반쯤 올라오는 리스트로 표시됩니다.
- 주유소 리스트는 보통휘발유 가격이 낮은 순으로 정렬하며, 거리와 브랜드, 주유소 이름, 유가를 함께 보여줍니다.
- 리스트 상단 손잡이를 드래그하거나 뒤로가기를 누르면 현재 상태에서 한 단계 아래 상태로 접힙니다.

## 최근 렌더링 메모

- 주유소 마커와 유가 말풍선은 서로 겹쳐도 뒤쪽 항목을 강제로 숨기지 않습니다.
- 화면 이동 중 깜박임을 줄이기 위해 라벨은 가능한 한 재사용하고, 화면 기준 우선순위만 다시 계산합니다.
- 주유소 라벨 경쟁은 끄고, 겹침 여부보다 화면 우선순위와 재사용 흐름을 우선합니다.
## 최근 반영 사항

- 주유소 리스트는 한 번 열리면 하단의 최소화된 상태로 남아 있다가, 헤더를 누르면 다시 펼칠 수 있습니다.
- 주유소 리스트의 제스처는 상단의 작은 가로 손잡이에서만 받도록 제한했습니다. 리스트 본문은 그대로 스크롤되고, 지도를 가릴 수 있는 영역을 최소화했습니다.
- 주유소 리스트의 실제 터치 영역은 상단의 손잡이보다 더 두껍고 가로로 길게 잡아, 가운데뿐 아니라 가장자리 쪽을 잡아도 드래그가 닿도록 했습니다.
- 주유소 리스트의 배경 스크림은 리스트가 반개 또는 최소화 상태일 때 터치를 가져가지 않도록 숨기고, 리스트 손잡이와 본문, 지도 터치가 서로 간섭하지 않도록 분리했습니다.
- 주유소 가격 박스는 줌 11~13에서 단계적으로 밝아지고, 줌 14~15에서 완전히 보이며, 줌 16~17에서는 더 큰 버전으로 표시됩니다.
- 주유소 가격 박스는 줌아웃할수록 서서히 투명해지고, 줌 10 이하에서는 사라집니다.
- 주유소 가격 박스는 흰색 배경과 검은 글자로 표시합니다.

## 최신 줌 규칙

- 줌 10에서는 주유소 아이콘만 보입니다.
- 줌 11과 12에서는 주유소 가격 박스가 단계적으로 더 밝아집니다.
- 줌 13부터 15까지는 주유소 가격 박스가 완전히 노출됩니다.
- 줌 16 이상에서는 더 큰 가격 박스 버전으로 자세히 표시됩니다.
- 가격 박스는 겹쳐도 어두워지지 않도록 흰색 배경을 유지하고, 투명도는 글자 중심으로만 조절합니다.

## UI 보조 규칙

- 주유소 리스트나 정보 패널이 보이는 동안에는 `현재 위치로 설정` 버튼을 숨기고, 패널이 완전히 숨겨졌을 때만 다시 나타납니다.

## 브랜드 로고

- 주유소 마커는 브랜드별 짧은 투명 로고 이미지를 사용합니다.
- 주유소 리스트는 좌측에 브랜드별 긴 로고 이미지를 표시합니다.
- 브랜드 로고는 크기 차이가 있어도 같은 기준 박스 안에 맞춰 표시하도록 정리했습니다.

### 브랜드 로고 파일

- 지도 마커: `drawable-nodpi/brand_*_short.png`
- 리스트 로고: `drawable-nodpi/brand_*_full.png`
- 로고 원본은 프로젝트 루트의 `logo/` 폴더에 보관합니다.
- 모든 로고는 배경 없이 투명 PNG를 그대로 사용합니다.

## 주유소 마커 및 가격 말풍선 규칙

- 주유소 로고는 기존보다 조금 더 작은 크기로 표시합니다.
- 유가 말풍선은 마커의 우상단에 더 가깝게 배치하고, 말풍선 꼬리가 주유소 마커를 가리키도록 표시합니다.
- 유가 말풍선은 완전 불투명하지 않게 약간의 투명도를 둡니다.
- 지도를 많이 줌아웃하면 일반 주유소 마커와 유가 말풍선은 사라지고, 최저가 상위 5개 주유소만 남깁니다.
- 유가 말풍선의 배경과 테두리도 투명도 값을 함께 반영합니다.

## 주유소 마커 및 유가 말풍선 조정

- 주유소 로고는 기존보다 조금 더 작게 표시합니다.
- 유가 말풍선의 꼬리는 중앙 하단을 기준으로 주유소 마커를 가리키도록 배치합니다.
- 유가 말풍선의 배경 투명도는 사용하지 않고, 말풍선 전체는 기본적으로 불투명하게 표시합니다.
- 지도를 많이 줌아웃하면 일반 주유소 마커는 더 빨리 사라지고, 최저가 상위 5개 주유소만 남습니다.
- 최저가 상위 5개 주유소는 유가 말풍선을 계속 유지합니다.
- 유가 말풍선의 글자 투명도는 줌아웃에 따라 더 빠르게 줄어듭니다.

## 주유소 마커 및 유가 말풍선 추가 규칙

- 지도가 충분히 줌아웃되면 top 5 주유소만 보입니다.
- 그보다 조금 더 줌인하면 top 20 주유소가 보입니다.
- 더 줌인하면 전체 주유소가 보입니다.
- top 5와 top 20 구간에서는 유가 말풍선을 완전 불투명보다 조금 약하게 보여 줍니다.
- top 5와 top 20 구간에서는 주유소 수를 줄여서 지도가 덜 가려지도록 합니다.

## 주유소 마커 및 유가 말풍선 추가 규칙

- 지도가 충분히 줌아웃되면 top 5 주유소만 보입니다.
- 그보다 조금 더 줌인하면 top 20 주유소가 보입니다.

## 주유소 리스트 뒤로가기 규칙

- 주유소 리스트가 떠 있는 상태에서 뒤로가기를 누르면 아예 닫지 않고 `전체 화면 -> 절반 -> 최소화` 순서로만 접힙니다.
- 일반 뒤로가기와 손잡이 제스처만으로는 리스트가 완전히 닫히지 않고, 최소화 상태로 남습니다.
- 리스트가 절반 상태일 때는 위로 쓸면 전체 화면으로 확장되고, 아래로 쓸면 최소화 상태로 접힙니다.
- 리스트가 전체 화면일 때는 아래로 한 번 쓸면 절반 상태로 돌아갑니다.
- 리스트가 전체 화면일 때 위로 스와이프해도 반응하지 않습니다.
- 뒤로가기는 전체 화면 -> 절반 -> 최소화 순서로만 동작하고, 최소화 상태에서는 더 이상 닫히지 않습니다.
- 최소화 상태에서 뒤로가기를 한 번 누르면 메인화면 이동 안내 토스트를 띄우고, 짧은 시간 안에 한 번 더 누르면 메인화면으로 나갑니다.
- 아래로 스와이프는 뒤로가기와 동일하게 동작합니다.
- 리스트 본문에서는 패널 상태 전환 제스처를 받지 않고, 항목 스크롤을 유지합니다.
- 절반 상태에서 아래로 스와이프하면 실제로 최소화 위치까지 이동합니다.
- 리스트 본문 스크롤은 그대로 살아 있고, 전체화면에서는 위로 스와이프해도 상태 변화가 없습니다.
